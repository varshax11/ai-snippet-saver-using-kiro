# AI Snippet Saver - Project Context

This Chrome extension was built using Kiro AI IDE to save selected text from ChatGPT and Gemini directly to Notion.

## Development Approach

- Used Kiro for rapid prototyping and iteration
- Leveraged AI assistance for API integration
- Real-time debugging with Kiro's error analysis

## Key Features Built with Kiro

1. Chrome extension scaffolding
2. Notion API integration
3. Context menu implementation
4. Message passing architecture
5. Error handling and user feedback
